# nba_win_prediction

# 🏀 NBA 플레이오프 우승팀 예측

XGBoost 득점 예측 모델 + 몬테카를로 시뮬레이션으로 2024-25 NBA 플레이오프 브래킷을 예측한 프로젝트.
**우승팀(OKC)·파이널 매치업·시리즈 스코어(4-3)까지 정확 적중**

## 파일 구성

| 파일 | 내용 |
| --- | --- |
| `[노트북 파일명 1]` | NBA API 데이터 수집 · 전처리 · 피처 엔지니어링 |
| `[노트북 파일명 2]` | 앙상블 4종 비교(XGBoost 선정) → 몬테카를로 시뮬레이션 |

## 기술 스택

`Python` `scikit-learn` `XGBoost` `LightGBM` `CatBoost` `nba_api`
